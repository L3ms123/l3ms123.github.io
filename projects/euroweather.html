EuroWeather DynamoDB - Real-time City Weather

  • Tech Stack: HTML, CSS, JavaScript, Boto3, AWS DynamoDB, API Gateway

Real-time weather display system for European cities and GPS coordinates using AWS DynamoDB as serverless backend.

Single-table design with composite keys (PK: geohash, SK: timestamp) storing temperature, humidity, wind speed, precipitation data with 7-day TTL retention.

Optimized for future Gradient Boosting predictions using 7 days historical data (~672 samples/city) for feature engineering (temp trends, precip patterns, volatility).

Cities dataset sourced from cities.json (CC BY 4.0). Supports both city name and lat/lon coordinate queries.
